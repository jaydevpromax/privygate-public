# PrivyGate Static Web Demo

This directory contains a dependency-free static demo for the 0G APAC Hackathon recording and public repository.

Open locally:

```powershell
Start-Process .\app\web\index.html
```

Or open the file directly in a browser:

```text
app/web/index.html
```

The page demonstrates the same scenario as `scripts/privygate_cli.py`:

- Alice satisfies `University:role:student AND Lab:role:researcher` and is accepted.
- Bob only has `University:role:student` and is rejected.
- Alice is rejected after the researcher credential is revoked.
- 0G Chain is presented as the registry and audit-evidence layer.
- 0G Storage is presented as the audit-manifest persistence layer.

The web page is static and does not perform real cryptographic verification in the browser. The executable prototype remains in `src/privygate/` and `scripts/privygate_cli.py`.

Hosted deployment:

- The public repository can deploy this directory with GitHub Pages.
- The workflow is `.github/workflows/pages.yml`.
- Expected public URL after Pages succeeds: `https://jaydevpromax.github.io/privygate-public/`.
- Verify the URL in a private browser window before adding it to project pages.

The storage manifest shown by the page is generated by:

```powershell
$env:PYTHONPATH='src'
python .\scripts\generate_storage_audit_manifest.py
```
